01 / Capabilities

从选区到成果，流程不再断开

GeoD 把分散的地图数据获取流程收进一个桌面工作台。无需在多个脚本和工具之间来回切换，下载任务、空间边界与导出结果始终保持关联。

10+ 内置图源

OSM、ArcGIS、天地图、Google Maps、高德地图等，并支持自定义 XYZ 瓦片服务。

XYZ / WMTS / CUSTOM SOURCE

3D Tiles 下载

支持 Cesium Ion、Google 3D Tiles 与自定义 URL，按空间范围递归下载并离线预览。

CESIUM / GOOGLE / LOD

历史影像回溯

通过 Esri World Imagery Wayback 时间轴定位历史时期影像，并按选区批量下载。

ESRI WAYBACK / TIMELINE

GeoTIFF 导出

自动写入坐标投影标签，支持 LZW / Deflate 压缩与大范围 BigTIFF 流式写入。

GEOTIFF / BIGTIFF / COMPRESS

多种空间选区

行政区划联动、手绘范围或导入 GeoJSON、Shapefile、KML、KMZ，按真实边界裁剪。

BOUNDARY / VECTOR IMPORT

可靠的大任务执行

多任务并行、断点续传与失败重试，缓存目录可迁移，让长期任务更稳定、更可控。

RESUME / RETRY / CACHE MOVE
