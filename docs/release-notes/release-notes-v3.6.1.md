# v3.6.1 - 修复自动更新检测

本版本修复部分网络环境下无法获取 GitHub Release 信息、导致自动更新检测失败的问题。

## 更新检测

- 更新信息改由桌面端后端请求，避免 WebView 的跨域限制和网络实现差异。
- 稳定版优先读取国内镜像站更新清单，镜像不可用时自动回退 GitHub API。
- Windows 安装包更新地址优先使用国内镜像，降低下载失败概率。
- 发布流程自动生成并同步 `update.json`，版本信息与安装包保持一致。

## 升级说明

- v3.6.0 用户可通过自动更新升级到后续版本。
- v3.5.3 及更早版本仍只会访问 GitHub API；如果检查更新失败，需要从官网或群文件手动安装一次新版。
