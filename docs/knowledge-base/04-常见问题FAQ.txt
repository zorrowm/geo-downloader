GeoDownloader 常见问题 FAQ
===========================

Q1: 软件收费吗？
A: 完全免费开源，MIT 协议，个人和商业用途都可以使用。

Q2: 安装时 Windows 提示"未知发布者"怎么办？
A: 这是 Windows SmartScreen 对未签名程序的正常提示。点击"更多信息"→"仍要运行"即可。软件是开源的，代码公开透明，不含任何恶意代码。

Q3: 下载速度很慢怎么办？
A: 可以尝试以下方法：
1. 调高并发数（设置中可改为 50-100）
2. 更换数据源（不同数据源服务器速度不同）
3. 国外数据源建议配置代理
4. 避免同时开太多下载任务

Q4: 下载的图片打开是黑色/白色的？
A: 可能是该区域在所选层级没有数据。建议：
1. 降低层级（比如从 18 降到 15）
2. 换其他数据源试试
3. 检查区域是否在数据源覆盖范围内

Q5: 能下载 Google 卫星影像吗？
A: 可以，内置了 Google Satellite 数据源。但需要注意：
1. 国内网络可能无法直接访问 Google 服务器
2. 需要在设置中配置代理（如 127.0.0.1:7890）
3. 下载仅供个人学习研究使用

Q6: 天地图需要 Token 吗？
A: 软件内置了默认 Token 可以直接使用。如果遇到限制，可以在天地图官网注册账号申请自己的 Token，在设置中替换。

Q7: 导出的 GeoTIFF 在 QGIS 中显示坐标不对？
A: 导出的 GeoTIFF 使用 EPSG:3857（Web Mercator）投影。如果需要其他坐标系，可以在 QGIS 中进行投影转换。

Q8: 能下载多大的区域？
A: 理论上没有面积限制，但实际受限于：
1. 磁盘空间
2. 内存配置（设置中 memory_budget_mb 可调到 16GB）
3. 层级越高、区域越大，瓦片数量指数增长
建议先小范围测试，确认没问题再下载大范围。

Q9: 下载中途可以暂停/继续吗？
A: 可以。点击任务的暂停按钮即可暂停，再次点击继续下载。即使关闭软件重新打开，也能继续。

Q10: 支持同时下载多个任务吗？
A: 支持。可以创建多个下载任务并行执行，每个任务独立管理。

Q11: 缓存占用太多磁盘空间怎么办？
A: 可以在设置中：
1. 设置缓存大小上限（tile_cache_max_size_mb）
2. 手动清除缓存
3. 关闭浏览缓存功能（tile_cache_enabled 设为 false）

Q12: macOS 提示"无法打开，因为无法验证开发者"？
A: 在系统偏好设置→安全性与隐私→通用中，点击"仍要打开"。

Q13: 3D Tiles 下载需要什么信息？
A: 需要以下任一：
1. Cesium Ion Asset ID + Access Token（在 Cesium Ion 账户中获取）
2. Google 3D Tiles API Key
3. 自定义 tileset.json 的完整 URL

Q14: 如何更新到最新版本？
A: 软件启动时会自动检测新版本并提示更新。也可以手动从官网或 GitHub 下载最新安装包覆盖安装。

Q15: 下载失败显示"部分瓦片缺失"怎么办？
A: 这表示部分区域的瓦片获取失败（可能是服务器限流或网络问题）。可以：
1. 点击"重试缺失"按钮只下载失败的瓦片
2. 点击"强制导出"按钮导出已有数据（缺失部分显示为白色）
3. 调低并发数后重新下载

Q16: 软件支持命令行/批量下载吗？
A: 目前是纯 GUI 桌面应用，暂不支持命令行模式。批量下载可以通过导入多要素的 Shapefile 实现。

Q17: 下载的数据可以商用吗？
A: 软件本身是 MIT 开源协议。但下载的数据版权归数据源所有（如 Google、ArcGIS 等各有使用条款），商用请确认数据源的许可协议。

Q18: 能下载历史卫星影像吗？
A: 可以，使用 Wayback 模块。它接入了 Esri 的 World Imagery Wayback 服务，覆盖 2014 年至今的历史卫星影像。支持按日期范围筛选。

Q19: 导出 MBTiles 后怎么使用？
A: MBTiles 文件可以：
1. 用 QGIS 直接打开查看
2. 部署为本地瓦片服务器（如 TileServer GL）
3. 在移动端 App 中作为离线地图包
4. 导入到 Mapbox Studio

Q20: 怎么加入交流群？
A: 关注公众号"GeoD"，回复"加群"获取最新微信群二维码。群二维码每周更新。
