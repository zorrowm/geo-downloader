GeoDownloader 版本更新记录
============================

v3.5.3（2026-05-29，当前最新版）
- 修复流式导出时坐标越界导致的程序崩溃
- 修复 BigTIFF 单条带标签内联问题
- 修复暂停/继续下载时的并发竞争条件
- 修复 JSON 持久化文件在崩溃时损坏的问题（改用原子写入）
- 修复前端部分场景的无限渲染循环

v3.5.2（2026-05-28）
- 新增实时无数据瓦片计数显示
- 新增启动时自动清理超过 24 小时的临时目录
- 缓存写入失败超过阈值时显示警告

v3.5.1（2026-05-28）
- 修复流式导出的内存溢出问题
- 修复浏览缓存的 CORS 跨域问题
- 修复 Token 刷新机制

v3.5.0（2026-05-20）— 重大更新
- 原始瓦片直接写入目标目录，跳过临时中转（减少磁盘IO）
- 多条带并行导出流水线（显著提升大区域导出速度）
- Wayback 支持按发布日期范围过滤历史版本
- 浏览缓存的瓦片自动从下载队列中跳过（避免重复下载）
- 新增会话状态恢复（重启后恢复上次的标签页和参数）

v3.4.5（2026-05-11）
- 缓存查询性能突破：批量 SQL 查询，速度提升约 3000 倍
- 零拷贝缓存命中：内存共享避免临时文件IO
- 部分失败自动导出：可配置成功率阈值，失败瓦片显示为白色
- 下载大小估算精度提升 17 倍
- 修复 Shapefile 单文件导入问题
- 42 项后端单元测试全部通过

v3.4.0（2026-04 月）— 大版本重构
- 前端完全迁移到 React 19 + TypeScript + shadcn/ui + Tailwind CSS v4
- 新增浏览即缓存模块：地图浏览时自动缓存瓦片
- 3D Tiles 增强：Cesium Ion 自动预览、3D 表面拾取、本地离线预览
- 矢量瓦片支持：MBTiles/GeoPackage 导出
- 区域选择支持 Shapefile、GeoJSON、KML、KMZ 多格式导入
- Wayback 时间轴重新设计，支持年份分组

v3.3.x（2026-02 月）
- 新增 Wayback 历史影像批量下载
- 新增 DEM 高程数据下载模块

v3.2.x（2026-01 月）
- TIFF 压缩三选一（无压缩/LZW/Deflate）
- 任务日志系统
- 断点续传增强
- BigTIFF 兼容性修复

v3.0.0（2025 年底）
- 产品更名为 GeoDownloader
- 新增 3D Tiles 下载与离线预览
- 全新界面设计
