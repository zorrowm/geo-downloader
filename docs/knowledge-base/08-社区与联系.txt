交流渠道：

2. 微信技术交流群：关注公众号回复"加群"获取最新群二维码（每周更新）
3. GitHub Discussions：https://github.com/gaopengbin/geo-downloader/discussions（使用问题、功能建议、案例分享）
4. GitHub Issues：https://github.com/gaopengbin/geo-downloader/issues（Bug 报告）

如何反馈 Bug：
1. 推荐方式：在 GitHub Issues 中提交，附带以下信息：
   - 操作系统和版本
   - GeoDownloader 版本号
   - 复现步骤
   - 错误截图或日志
2. 也可以在微信交流群中反馈

如何提建议：
- 在 GitHub Discussions 中发帖讨论
- 或在微信交流群中提出
