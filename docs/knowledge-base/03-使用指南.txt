GeoDownloader 使用指南
======================

一、下载安装

Windows 用户：
1. 从官网 https://geodownloader.pages.dev 或国内高速下载 https://laogao.xyz/packages/latest/ 下载 .exe 安装包
2. 双击运行安装（Windows SmartScreen 可能提示未知发布者，点击"仍要运行"即可）
3. 安装完成后从桌面或开始菜单启动

macOS 用户：
1. 下载对应芯片版本的 .dmg 文件（arm64 适用 M1/M2/M3/M4，x64 适用 Intel）
2. 双击 .dmg，将应用拖到 Applications 文件夹
3. 首次打开可能需要在"系统偏好设置→安全性"中允许运行

Linux 用户：
- Debian/Ubuntu：下载 .deb 文件，执行 sudo dpkg -i GeoDownloader_xxx.deb
- 其他发行版：下载 .AppImage 文件，chmod +x 后直接运行

二、基本下载流程

1. 选择数据源：在左侧面板选择要下载的地图类型（影像/矢量/3D/Wayback）
2. 选择区域：
   - 手绘矩形/多边形：在地图上直接框选
   - 行政区划：选择省→市→区
   - 导入文件：支持 Shapefile(.zip)、GeoJSON、KML、KMZ
   - 搜索地名：输入地名自动定位
3. 设置参数：
   - 层级范围（如 1-18 级）
   - 并发数（默认 30）
   - 导出格式（GeoTIFF/PNG/JPEG/MBTiles 等）
4. 点击"开始下载"
5. 下载完成后可在任务列表中查看结果，点击打开输出目录

三、导出格式说明

GeoTIFF (.tif)：
- 标准 GIS 格式，带地理参考信息
- 支持 LZW/Deflate 压缩
- 可直接用 QGIS、ArcGIS 等软件打开
- 适合做学术分析、打印地图

PNG (.png)：
- 无损压缩，支持透明度
- 适合网页展示、快速预览

JPEG (.jpg)：
- 有损压缩，文件体积最小
- 适合对文件大小敏感的场景

MBTiles (.mbtiles)：
- SQLite 容器格式
- 适合移动端离线地图、Mapbox 样式展示

GeoPackage (.gpkg)：
- OGC 标准格式
- 适合与 QGIS 等桌面 GIS 配合使用

原始瓦片目录：
- 保持 {z}/{x}/{y} 目录结构
- 适合自建瓦片服务器

四、区域选择方式

1. 手绘矩形：点击工具栏矩形按钮，在地图上拖动框选
2. 手绘多边形：点击多边形按钮，依次点击顶点，双击结束
3. 行政区划：省→市→区三级下拉，自动加载边界
4. 导入文件：
   - Shapefile：支持 .zip 压缩包或 .shp+.shx+.dbf 三件套
   - GeoJSON：标准 GeoJSON 文件
   - KML/KMZ：Google Earth 格式
5. 地名搜索：输入地名关键字自动补全定位

五、高级功能

浏览即缓存：
- 在地图上浏览时，看过的瓦片自动缓存到本地
- 下载时如果缓存中已有对应瓦片，直接复用不再重复下载
- 可在设置中配置缓存大小上限
- 可清除指定数据源或层级的缓存

断点续传：
- 下载过程中如果软件崩溃或关闭
- 下次启动时自动恢复未完成的任务
- 已下载的瓦片不会重复下载

多边形裁剪：
- 选择不规则区域时，导出的 GeoTIFF 会自动裁剪
- 区域外部分显示为透明

并发控制：
- 默认 30 并发，可在 10-100 之间调整
- 并发太高可能被服务器限制，建议根据数据源调整

六、3D Tiles 专项

1. 输入 Cesium Ion Asset ID 和 Token
2. 软件自动加载 3D 场景预览
3. 在 3D 场景中直接框选下载区域（表面拾取）
4. 下载后支持本地离线预览
5. 可调节模型精度（SSE）、透明度、坐标偏移

七、Wayback 历史影像

1. 选择 Wayback 数据源
2. 软件扫描所有可用历史版本
3. 在时间轴中选择需要的年份/日期
4. 支持按发布日期范围过滤
5. 下载选中时间点的历史卫星影像
